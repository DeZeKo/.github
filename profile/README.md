# DeZeKo

DeZeKo is a small software organization run by two friends building projects together.

We use this space to collect, develop, and create things we find interesting, from useful tools to experiments that may or may not turn into something bigger.

---

## Who we are

DeZeKo is maintained by:

- ZennDC ([GitHub](https://github.com/ZennDC))
- KobeD62 ([GitHub](https://github.com/KobeD62))

We work together on ideas, split development, and occasionally rebuild things from scratch when we change our minds.

---

## What this is

This org is basically our shared workspace.

You’ll find:

- Game engines
- Utilities and small tools
- School projects
- Experimental side projects
- Late-night ideas that somehow became code

Some projects are serious. Some are not. Most are somewhere in between.

---

## Tech we use

It changes depending on the project, but usually includes:

- Java
- C/C++
- Python
- Whatever else makes sense at the time

We don’t really stick to one stack, we use what gets the job done.

---

## How we build

A few things we generally try to follow:

- Keep things simple when possible
- Build first, refine later
- Don’t overthink early versions
- Learn as we go
- Rework things when we feel like it

---

## Projects

Repos here are either:

- actively being worked on
- finished (rare, but it happens)
- or abandoned until further notice

Feel free to look around.
